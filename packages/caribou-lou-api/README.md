# caribou-lou-api

This is the module that allows caribou-lou-bot to retrieve the information about the resume.

## Enviroment

|                |Value                          |Default                         |
|----------------|-------------------------------|-----------------------------|
|DB_URL          |It's the url to the database   |mongodb://ciervo:27017/resume            |

## Usage

To start the api just run the next sentence:
>`npm run start`
